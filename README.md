# book-appointment-component
Front end React component that allows a 3rd party website to add appointment/event booking functionality.

## Install
Run 'npm install'

## How to run
1. Run 'npx http-server -c-1'. This will run a local http server that can be accessed on http://localhost:8080
2. Obtain an integrationId value and pass it as query param to http://localhost:8080?integrationId=\<value\>
