# book-appointment-component
Front end React component that allows a 3rd party website to add appointment/event booking functionality.

## Install
Run 'npm install'

## How to run
1. Edit /public/index.html and replace integrationId with an appBooking integration id
2. Run 'npx http-server -c-1'. This will run a local http server that can be accessed on http://localhost:8080
